API Яндекс.Карт позволяет отображать на страницах сайта панорамы Яндекса.

Панорама реализуется объектом . Этот объект содержит необходимую информацию о панораме: ее название, географические координаты, связи со смежными панорамами и др.

Для добавления панорамы на страницу предназначен объект (далее плеер). Плеер формирует запросы за тайлами и создает на странице элемент <canvas> для отображения панорамы. Чтобы вставить панораму на страницу, достаточно передать конструктору плеера объект панорамы (Panorama).

Получить объект панорамы можно с помощью функции . На вход ей подаются географические коордианты точки, для которой нужно получить панораму. Функция возвращает массив панорам, найденных в окрестностях заданной точки. Панорамы в массиве отсортированы по расстоянию.

Также для добавления панорамы на страницу можно воспользоваться функцией . На вход ей подаются коордианты точки, в которой нужно открыть панораму. Функция выполнит поиск ближайшей панорамы и в случае успеха создаст плеер с найденной панорамой.
