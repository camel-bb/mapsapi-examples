The API allows you to add placemarks with glyph icons (Glyph) to the map.

To create a placemark with a glyph icon, the value of the "preset" option should prefixed with "glyph" (for example, "islands#glyphCircleIcon"). Use the "iconGlyph" option to specify the name of a glyph icon.

Note that this example requires Bootstrap.
